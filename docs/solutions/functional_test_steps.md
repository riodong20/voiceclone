**版本:** v0.1.0.0

**适用范围:** Web UI 自动化测试

## 前置条件
1. 后端服务已启动: `cd backend && uv run uvicorn main:app --host 127.0.0.1 --port 8002`
2. 前端服务已启动: `cd frontend && npm run dev` (端口 5173)
3. 测试资源准备:
   - 有效音频文件: `test_audio.wav` (16bit, 16kHz, 单声道, 5-30秒)
   - 无效格式文件: `test_file.txt`, `test_image.jpg`
   - 超长音频文件: > 10MB

## 测试模块 2: 声音克隆功能
### 测试用例 2.1: 音频文件上传 - 成功场景
**操作步骤:**
1. 导航到 "克隆声音" 页面
2. 点击 "上传音频文件" 按钮
3. 选择有效 WAV/MP3/OGG/WebM 格式的音频文件 (5-30秒)
4. 在弹出的对话框中输入声音名称: "测试声音"
5. 点击 "开始克隆" 按钮
6. 等待克隆完成 (最多等待 30秒)

**预期结果:**
✅ 上传进度条显示正常，百分比从 0% 到 100%
✅ 克隆成功提示出现，包含 "克隆完成" 文字
✅ 声音列表中出现新创建的声音条目，名称为 "测试声音"
✅ 点击该声音条目，可以看到声音详情，包含创建时间、时长等信息

### 测试用例 2.2: 音频文件上传 - 失败场景
**操作步骤:**
1. 导航到 "克隆声音" 页面
2. 点击 "上传音频文件" 按钮
3. 选择非音频格式文件 (如 txt/jpg)
4. 点击 "开始克隆" 按钮

**预期结果:**
✅ 立即显示错误提示: "不支持的文件格式，请上传 MP3/WAV/OGG/WebM 格式的音频"
✅ 页面无崩溃，仍可正常操作

### 测试用例 2.3: 浏览器录音 - 成功场景
**操作步骤:**
1. 导航到 "克隆声音" 页面
2. 点击 "开始录音" 按钮
3. 授权浏览器访问麦克风权限
4. 录制 5-10秒语音后点击 "停止录音"
5. 预览录制的音频，点击 "确认使用"
6. 输入声音名称: "录音测试"，点击 "开始克隆"
7. 等待克隆完成

**预期结果:**
✅ 录音过程中波形动画正常显示
✅ 停止后可以正常播放录制的音频
✅ 克隆成功，声音列表中出现 "录音测试" 条目

### 测试用例 2.4: 浏览器录音 - 取消权限场景
**操作步骤:**
1. 导航到 "克隆声音" 页面
2. 点击 "开始录音" 按钮
3. 在浏览器权限请求中点击 "拒绝"

**预期结果:**
✅ 显示错误提示: "需要麦克风权限才能进行录音，请在浏览器设置中允许访问"
✅ 页面功能正常，可继续其他操作

## 测试模块 4: TTS 语音合成功能
### 测试用例 4.1: 标准声音合成 - 成功场景
**操作步骤:**
1. 导航到 "语音合成" 页面
2. 在文本框中输入: "你好，这是语音合成测试"
3. 选择标准声音 (如云溪/小刚)
4. 调整语速为 1.0，音量为 80，音调为 0
5. 点击 "开始合成" 按钮
6. 等待合成完成

**预期结果:**
✅ 合成进度显示正常
✅ 合成成功后音频自动播放或显示播放按钮
✅ 音频内容与输入文本一致
✅ 合成记录出现在历史列表中

### 测试用例 4.2: 克隆声音合成 - 成功场景
**操作步骤:**
1. 导航到 "语音合成" 页面
2. 在文本框中输入: "你好，这是克隆声音合成测试"
3. 选择之前克隆成功的自定义声音
4. 点击 "开始合成" 按钮
5. 等待合成完成

**预期结果:**
✅ 合成成功，音频可正常播放
✅ 声音特征与克隆的源声音一致
✅ 合成历史中显示该记录

### 测试用例 4.3: TTS 合成 - 边界场景
**操作步骤:**
1. 导航到 "语音合成" 页面
2. 输入空文本，点击 "开始合成"
3. 输入超过 1000 字符的长文本，点击 "开始合成"
4. 调整语速为 0.4 (低于最小值)，点击 "开始合成"
5. 调整语速为 2.1 (高于最大值)，点击 "开始合成"
6. 调整音量为 -1 或 101，点击 "开始合成"
7. 调整音调为 -13 或 13，点击 "开始合成"

**预期结果:**
✅ 所有非法输入均显示对应的错误提示
✅ 合成操作不会执行
✅ 页面无崩溃，可继续操作

## 测试模块 5: 时间轴功能
### 测试用例 5.1: 时间轴基础操作
**操作步骤:**
1. 导航到 "时间轴" 页面
2. 点击 "新建项目" 按钮，输入项目名称 "测试项目"
3. 点击 "添加片段" 按钮，输入文本 "片段1"，设置开始时间 0s，结束时间 2s
4. 点击 "添加片段" 按钮，输入文本 "片段2"，设置开始时间 2s，结束时间 4s
5. 拖拽片段调整位置
6. 点击 "保存项目" 按钮

**预期结果:**
✅ 项目创建成功，时间轴显示空轨道
✅ 片段添加成功，显示在对应时间位置
✅ 拖拽操作正常，片段位置更新
✅ 项目保存成功，刷新页面后仍可看到该项目

### 测试用例 5.2: 批量合成功能

4. 点击 "开始批量合成"
5. 等待合成完成

**预期结果:**
✅ 所有片段依次合成，进度显示正常
✅ 合成完成后每个片段都有对应的音频
✅ 可以单独播放每个片段的音频
✅ 时间轴上每个片段显示音频波形

### 测试用例 5.3: 时间轴导出功能
**操作步骤:**

**预期结果:**
